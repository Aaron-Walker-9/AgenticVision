<h1 align="center"> AgenticVision Version 1.0</h1> 

<p align="center">
As agentic AI applications proliferate, network operators like BT face challenges in capacity planning and service differentiation. This project develops a privacy-preserving ML system to detect and quantify AI workload patterns in encrypted traffic, enabling operators to track AI adoption, forecast resource demand and design AI-optimized service products.
