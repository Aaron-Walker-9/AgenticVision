<h1 align="center"> AgenticVision Version 1.0</h1> 

<p align="center">
As agentic AI applications proliferate, network operators face challenges such as AI capacity planning and new product and service differentiation. This repo contains a privacy-preserving ML model called AgenticVision used for the detection and quantification of AI workload patterns in encrypted traffic, enabling operators to track AI adoption, forecast resource demand and design AI-optimized service products.
